BREAKING — Chrome Extension Relay removido
Módulo 6 · Browser / Ferramentas
v3.22 ⚠ Breaking
O driver: "extension" e o browser.relayBindHost foram removidos. Quem configurou o browser via Chrome Relay Extension precisa migrar para existing-session ou user. Rodar openclaw doctor --fix faz a migração automática.
Config do aluno — Antes → Depois
browser: profiles: default: driver: "extension" relayBindHost: "127.0.0.1"
→
browser: profiles: user: driver: "existing-session" # Migração automática: openclaw doctor --fix
Arquivo: aula-06-browser/prompts/ (Drive) — se existir menção ao Chrome Relay
💥
BREAKING — Geração de imagem: nano-banana-pro removido
Módulo 5 · Skills & Tools
v3.22 ⚠ Breaking
A skill bundled nano-banana-pro para geração de imagens foi removida. O caminho agora é usar agents.defaults.imageGenerationModel.primary no config. Exemplo: "google/gemini-3-pro-image-preview". Quem copiou o sample-skill config antigo precisa atualizar.
Config do aluno — Antes → Depois
skills: nano-banana-pro: enabled: true config: ...
→
agents: defaults: imageGenerationModel: primary: "google/gemini-3-pro- image-preview"
Arquivo: PRDs de skills (se mencionam nano-banana-pro)

v3.22 Prompt + PRD

Modelo OpenAI padrão: GPT-5.4
Módulo 3 · Modelos & Configuração
v3.22 Prompt + PRD
Quem configura OpenAI como provider agora recebe GPT-5.4 por padrão (antes era GPT-4.5). Variantes gpt-5.4-mini e gpt-5.4-nano também disponíveis. Impacto em custos — pode ser mais caro sem configurar tier adequado. Material de modelos precisa atualizar a tabela de preços e recomendações.
Material do módulo — Atualizar tabela

→
Modelos OpenAI recomendados: - GPT-5.4 (default) - GPT-5.4-mini (economia) - GPT-5.4-nano (ultra-low-cost) - o3 (reasoning) ⚠️ Custo: verificar pricing do GPT-5.4 vs GPT-4.5
Arquivo: aula-03-modelos/prompts/ (Drive)

Per-agent thinking/reasoning/fast defaults
Módulo 8 · Multi-Agentes
v3.22 Feature nova
Cada agente agora pode ter defaults próprios de thinking, reasoning e fast mode. Se um agente tenta mudar para um modelo não permitido, o sistema reverte automaticamente para o default do agente. Melhora o controle de custos em setup multi-agente — material do Módulo 8 precisa cobrir isso.
Config exemplo — Novo recurso
agents: meu-agente: model: "anthropic/claude-sonnet" # Sem controle de thinking
→
agents: meu-agente: model: "anthropic/claude-sonnet" thinking: "on" reasoning: "stream" fast: "openai/gpt-5.4-mini" # Override auto-revertido se # modelo não permitido
