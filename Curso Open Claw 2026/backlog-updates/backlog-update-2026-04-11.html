Contexto

Em 04/04/2026, a Anthropic bloqueou OAuth para planos Pro. O curso precisou migrar de Anthropic como provider padrão para ChatGPT (OpenAI) via OAuth.

Este ciclo de atualização cobre:

  • Migração completa de OAuth: Anthropic → ChatGPT OAuth
  • Modelo padrão: claude-sonnet-4-5 → openai/gpt-5.4
  • Modelo heartbeat: claude-haiku-4-5 → openai/gpt-4o-mini
  • Novas features: Dreaming (M4), ACP Bind (M8), Secrets Audit (M9)
  • Infraestrutura: KVM 1 → KVM 2, cupom BRUNOOKAMOTO, SSH key setup
