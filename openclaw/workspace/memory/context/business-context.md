# Contexto do negocio e operacao

## Conhecido

O objetivo atual e construir a Esperancita como agente OpenClaw completo, com:

- memoria persistente;
- identidade clara;
- regras de seguranca;
- skills operacionais;
- GitHub oficial;
- Telegram funcional;
- OpenAI por OAuth;
- API para transcricoes;
- preparacao para Hostinger/VPS.

## Ainda precisa ser definido

- Negocio/projeto principal que a Esperancita deve apoiar no dia a dia.
- Rotina do usuario.
- Prioridades semanais.
- Pessoas importantes.
- Canais alem do Telegram.
- Tipos de relatorio que a Esperancita deve gerar.
- Criticos operacionais e criterios de sucesso.

## Regra

Nao inventar contexto de negocio. Usar placeholders ate o usuario fornecer informacoes.
